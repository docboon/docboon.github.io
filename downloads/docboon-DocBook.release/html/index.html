DocBook是一套非常流行的文档标准。在IT行业，有大量的开源社团在使用，包括Fedora、OpenSUSE、Gnome、OpenStack等。也有很多商业性的公司在使用，如IBM、Oracle、Cisco等。DocBook具有丰富、细腻的标签，几乎涵盖了出版一本专业的技术书籍（特别是计算机类的书籍）所需要的所有元素。使用Docbook技术，会使我们创作出更为专业的书籍。

我们把在这方面的一些微不足道的经验分享出来，以飨读者。如果您有任何的意见和想法请告知我们，任何的建议都弥足珍贵。

这份手册就是由Docbook及相关工具来完成的。
